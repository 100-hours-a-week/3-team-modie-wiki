## 프로젝트 소개

- **이름:** MODiE
- **목표:** 일회성 모임을 쉽고 빠르게 모집하고, 일정 조율부터 정산까지 한 번에 해결하는 서비스 🙌🏻

<br />

## 주요 기능

- 모임 개설 / 참여 / 삭제
- 실시간 채팅 기능
- 푸시 알림(PWA 기반)
- 사용자 카카오 로그인/회원가입
